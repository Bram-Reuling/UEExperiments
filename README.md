# UEExperiments

Repository where I experiment with things within Unreal Engine. 

## Usage
This project has been made with Unreal Engine 5.7.3. You will need to have MSVC installed to compile the source code that comes with the project.

## Contributing
I do not accept any PR's to this repository since this is basically a playground for me. I will run tests, experiment with gameplay idea's and build tooling in this repository that I will use later again.

Feel free to use snippets of code that you find in here!
